Predictive Maintenance System

Real-time Machine Health Monitoring & Failure Prediction

Project Overview

The IoT Predictive Maintenance System analyzes machine sensor data such as temperature, pressure, and vibration to predict potential machine failures before they occur. Using Machine Learning algorithms and Explainable AI (SHAP), the system detects patterns in sensor readings and calculates the probability of machine failure. This allows industries to perform maintenance before breakdowns occur, reducing downtime and improving operational efficiency.

The system also provides visual insights into predictions using SHAP explainability plots, enabling engineers to understand which features influence failure risk the most.

Key Features

📡 Real-Time Sensor Monitoring

Monitor machine parameters like temperature, pressure, and vibration.

🤖 Machine Learning Prediction

Predict machine failure probability using trained ML models.

📊 Interactive Dashboard

Visualize predictions and machine health through interactive charts.

🧠 Explainable AI (SHAP)

Understand which features influence the model's prediction.

⚙️ Maintenance Recommendations

Receive maintenance suggestions based on prediction results.

📜 Prediction History

Track previous predictions and sensor readings.
